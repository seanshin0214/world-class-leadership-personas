업로드 방법:
============
1. ChatGPT 웹사이트 로그인 (https://chatgpt.com)
2. "Explore GPTs" → "My GPTs" → "Persona Hub - 108 Expert AI Personas" 선택
3. "Edit" 버튼 클릭
4. Knowledge 섹션에서 기존 파일 모두 삭제
5. 이 폴더의 20개 .txt 파일 모두 업로드
6. "Update" 또는 "Save" 클릭
