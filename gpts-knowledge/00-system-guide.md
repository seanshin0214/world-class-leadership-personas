# Leader's Decision Assistants - System Guide

## Overview

Leader's Decision Assistants is a comprehensive expert consultation system featuring **142 World-Class+ certified expert personas** across 9 professional domains. Each expert provides evidence-based insights, cross-disciplinary thinking, and practical solutions.

---

## How to Use

### Basic Expert Call
Simply describe your challenge, and the system will match you with the optimal expert:
- "I need help with system architecture"
- "Help me create a marketing strategy"
- "Review my code for security issues"

### Direct Expert Call
Call a specific expert by name or number:
- "@ai-engineer" or "@104"
- "Call the strategy consultant"
- "I want to talk to the CFO advisor"

### Multi-Expert Consultation
For complex problems, chain multiple experts:
- "First get input from the architect, then the security expert"
- "I need both marketing and finance perspectives"

## Best Practices

### For Strategic Decisions
1. Start with Strategy Consultant (303) or Strategic Oracle (326)
2. Add domain-specific experts as needed
3. Conclude with CFO Advisor (312) for financial impact

### For Technical Implementation
1. Begin with Systems Architect (128/129)
2. Engage relevant specialists (Frontend, Backend, DevOps)
3. End with Security Expert (126/127) and QA (501/502)

### For Product Development
1. Start with Product Manager (306) or Product Strategist (307)
2. Include UX Designer (201-204) and relevant engineers
3. Add Business Analytics (305) for market validation

### For Crisis Management
1. Lead with relevant domain expert
2. Add Legal Advisor (901) if needed
3. Include PR Specialist (322) for communications
