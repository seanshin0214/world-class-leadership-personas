## Brand Positioning Framework

### Positioning Statement Template

```markdown
FOR [target customer]
WHO [need/opportunity]
[BRAND] is a [category]
THAT [unique benefit]
UNLIKE [competitors]
WE [differentiator]

EXAMPLE - Mailchimp:
For small businesses
Who need marketing tools
Mailchimp is an email platform
That makes marketing easy and affordable
Unlike complex enterprise tools
We provide simplicity and personality
```

## Visual Identity System

### Logo Variations

```
PRIMARY LOGO: Full wordmark + icon
SECONDARY: Icon only (square spaces)
LOCKUP: Wordmark only (horizontal spaces)

USAGE RULES:
- Min size: 24px height
- Clear space: 2x logo height
- Never rotate, distort, or recolor
```

### Color System


## Brand Voice Guidelines

### Tone Spectrum

```
SITUATIONS:

CELEBRATORY (Product launch)
Tone: Excited, enthusiastic, fun
Example: "üéâ We did it! New features are LIVE!"

INFORMATIVE (Blog post)
Tone: Helpful, clear, friendly
Example: "Here's how to set up your first campaign."

EMPATHETIC (Error message)
Tone: Understanding, supportive
Example: "Oops! Looks like something went wrong. 
We're on it."

SERIOUS (Security issue)
Tone: Direct, transparent, calm
Example: "We take security seriously. Here's what 
happened and what we're doing."
```

### Writing Principles

```
‚úì DO:
- Use contractions (we're, you'll, it's)
- Write conversationally
- Be specific, not vague
- Use active voice
- Show personality

‚úó DON'T:
- Use jargon or buzzwords
- Be condescending
- Overuse exclamation points!!!
- Write in passive voice
- Be generic
```
