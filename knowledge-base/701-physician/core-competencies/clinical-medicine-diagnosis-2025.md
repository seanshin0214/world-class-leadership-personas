## Clinical Reasoning Framework

```
PATIENT ENCOUNTER STRUCTURE:

1. CHIEF COMPLAINT (1 min)
   "What brings you in today?"
   Document in patient's own words

2. HISTORY OF PRESENT ILLNESS (5-8 min)
   OLDCARTS mnemonic:
   - Onset: When did it start?
   - Location: Where is it?
   - Duration: How long does it last?
   - Character: Sharp, dull, burning?
   - Aggravating factors: What makes it worse?
   - Relieving factors: What makes it better?
   - Timing: Constant or intermittent?
   - Severity: 0-10 scale

3. REVIEW OF SYSTEMS (2-3 min)
   Constitutional: Fever, chills, weight loss, fatigue
   HEENT: Headache, vision changes, hearing loss
   Cardiovascular: Chest pain, palpitations, edema
   Respiratory: Cough, shortness of breath, wheezing
   GI: Nausea, vomiting, diarrhea, constipation
   GU: Dysuria, frequency, hematuria
   MSK: Joint pain, swelling, stiffness
   Neuro: Weakness, numbness, dizziness
   Psych: Mood, sleep, anxiety, depression
   Skin: Rash, lesions, changes

4. PAST MEDICAL HISTORY (1-2 min)
   - Chronic conditions (diabetes, hypertension, etc.)
   - Surgeries
   - Hospitalizations

5. MEDICATIONS & ALLERGIES (1 min)
   - Current medications (name, dose, frequency)
   - Over-the-counter medications
   - Supplements
   - Drug allergies (reaction type)

6. SOCIAL HISTORY (1-2 min)
   - Tobacco (pack-years)
   - Alcohol (drinks/week)
   - Recreational drugs
   - Occupation
   - Living situation
   - Sexual history (if relevant)

7. FAMILY HISTORY (1 min)
   - Parents: Age, health conditions, cause of death
   - Siblings: Major illnesses
   - Hereditary diseases

TOTAL TIME: 15-20 minutes for new patient
```

## Differential Diagnosis

### Chest Pain Workup

```markdown
RED FLAGS (Rule out immediately):
ðŸš¨ Acute MI
ðŸš¨ Pulmonary embolism
ðŸš¨ Aortic dissection
ðŸš¨ Tension pneumothorax
ðŸš¨ Pericardial tamponade

DIFFERENTIAL DIAGNOSIS (by system):

CARDIAC:
1. Acute Coronary Syndrome (ACS)
   - Symptoms: Chest pressure, left arm/jaw radiation, diaphoresis
   - Risk factors: Age >40, diabetes, hypertension, smoking, family history
   - Tests: ECG, troponin, CXR
   - Red flags: ST elevation, positive troponin

2. Pericarditis
   - Symptoms: Sharp chest pain, worse with inspiration, relieved by leaning forward
   - Tests: ECG (diffuse ST elevation), CXR
   - Treatment: NSAIDs, colchicine

PULMONARY:
3. Pulmonary Embolism (PE)
   - Symptoms: Sudden dyspnea, pleuritic chest pain, tachycardia
   - Risk factors: Recent surgery, immobilization, malignancy, hypercoagulable state
   - Tests: Wells score, D-dimer, CT angiography
   - Treatment: Anticoagulation

4. Pneumonia
   - Symptoms: Cough, fever, dyspnea, pleuritic pain
   - Tests: CXR (infiltrate), CBC (leukocytosis)
   - Treatment: Antibiotics

5. Pneumothorax
   - Symptoms: Sudden sharp chest pain, dyspnea
   - Exam: Decreased breath sounds, hyperresonance
   - Tests: CXR (air in pleural space)
   - Treatment: Chest tube if large

VASCULAR:
6. Aortic Dissection
   - Symptoms: Sudden tearing chest pain radiating to back
   - Exam: Blood pressure difference between arms, new murmur
   - Tests: CXR (widened mediastinum), CT angiography
   - Treatment: Emergency surgery

GI:
7. GERD
   - Symptoms: Burning chest pain, worse after meals, relieved by antacids
   - Tests: Clinical diagnosis, endoscopy if alarm symptoms
   - Treatment: PPI (omeprazole)

8. Esophageal spasm
   - Symptoms: Intermittent chest pain, dysphagia
   - Tests: Esophageal manometry
   - Treatment: Calcium channel blockers, nitrates

MSK:
9. Costochondritis
   - Symptoms: Sharp chest pain, reproducible with palpation
   - Exam: Tenderness over costochondral junctions
   - Treatment: NSAIDs

INITIAL WORKUP:

â–¡ Chest X-ray
â–¡ Troponin (x2, 3 hours apart)
â–¡ D-dimer (if PE suspected)
â–¡ CBC, BMP
â–¡ Lipid panel (if ACS)
```

### Diagnostic Algorithm (Chest Pain)

```python
def chest_pain_workup(patient):
    """
    Clinical decision support for chest pain
    """
    # Step 1: Assess for life-threatening causes
    if patient.has_st_elevation or patient.troponin_positive:
        return "ðŸš¨ STEMI - Activate cath lab immediately"
    
    if patient.sudden_onset and patient.tearing_pain and patient.bp_difference > 20:
        return "ðŸš¨ Aortic dissection - CT angiography STAT"
    
    if patient.wells_score_pe >= 4 or patient.d_dimer_elevated:
        return "ðŸš¨ PE likely - CT angiography STAT"
    
    # Step 2: Calculate HEART score for ACS risk
    heart_score = calculate_heart_score(patient)
    
    if heart_score <= 3:
        # Low risk (0-3): 1.7% chance of MACE at 6 weeks
        return "Low risk - Discharge with PCP follow-up"
    elif heart_score <= 6:
        # Moderate risk (4-6): 12% chance of MACE
        return "Moderate risk - Admit for observation, serial troponins"
    else:
        # High risk (7-10): 65% chance of MACE
        return "High risk - Admit to cardiology, likely cath"

def calculate_heart_score(patient):
    """
    HEART Score for chest pain risk stratification
    """
    score = 0
    
    # History
    if patient.highly_suspicious_for_acs:
        score += 2
    elif patient.moderately_suspicious:
        score += 1
    # else: 0 (slightly suspicious)
    
    # ECG
    if patient.ecg_significant_st_depression:
        score += 2
    elif patient.ecg_nonspecific_repolarization:
        score += 1
    # else: 0 (normal)
    
    # Age
    if patient.age >= 65:
        score += 2
    elif patient.age >= 45:
        score += 1
    # else: 0 (<45)
    
    # Risk factors (â‰¥3 of: HTN, DM, smoking, family history, hyperlipidemia, obesity)
    if patient.risk_factors >= 3:
        score += 2
    elif patient.risk_factors >= 1:
        score += 1
    # else: 0
    
    # Troponin
    if patient.troponin >= 3x_upper_limit:
        score += 2
    elif patient.troponin >= 1x_upper_limit:
        score += 1
    # else: 0 (normal)
    
    return score

# Example usage
```

## Clinical Guidelines (2025)

### Hypertension Management

```markdown
JNC 8 GUIDELINES:

BLOOD PRESSURE TARGETS:
- Age <60: <140/90
- Age â‰¥60: <150/90
- Diabetes/CKD: <140/90

FIRST-LINE MEDICATIONS:

1. ACE INHIBITOR (lisinopril 10-40mg daily)
   - Best for: Diabetes, CKD, heart failure
   - Side effects: Cough, hyperkalemia, angioedema
   - Monitor: Creatinine, potassium

2. ARB (losartan 50-100mg daily)
   - Alternative to ACE-I if cough
   - Same benefits, fewer side effects

3. CALCIUM CHANNEL BLOCKER (amlodipine 5-10mg daily)
   - Best for: Elderly, African American
   - Side effects: Edema, constipation

4. THIAZIDE DIURETIC (hydrochlorothiazide 12.5-25mg daily)
   - Cheap, effective
   - Side effects: Hypokalemia, hyperglycemia, hyperlipidemia
   - Monitor: Potassium, glucose

COMBINATION THERAPY:
- Start with 2 drugs if BP >160/100 or 20/10 above goal
- Combinations:
  * ACE-I + CCB
  * ACE-I + Thiazide
  * ARB + CCB
  * ARB + Thiazide

RESISTANT HYPERTENSION (not controlled on 3 drugs):
- Add spironolactone 25mg daily
- Check for secondary causes (renal artery stenosis, hyperaldosteronism, pheochromocytoma)
```

## Key Takeaways

1. **Think horses, not zebras** - Common things are common
2. **Rule out worst first** - Red flags = immediate workup
3. **Evidence-based** - Follow guidelines, not anecdotes
4. **Serial assessment** - Reassess if not improving
5. **Communicate clearly** - Explain diagnosis & plan in plain language
